#centos7下nginx高并发配置

##系统层优化

  1. 系统信息查看

    确认是64位还是32位:

    uname -a

    核数—>

    cat /proc/cpuinfo | grep name | cut -f2 -d: | uniq -c

    逻辑CPU信息:

    cat /proc/cpuinfo | grep physical | uniq -c

    CPU运行在多少位模式下:

    getconf LONG_BIT

    查看环境变量:

    env

    查看平均负载情况[除了前3个数字表示平均进程数量外，后面的1个分数，分母表示系统进程总数，分子表示正在运行的进程数；最后一个数字表示最近运行的进程ID]:

    cat /proc/loadavg

    内存的详细信息:

    cat /proc/meminfo

    查看所有网络接口的属性:

    ifconfig

    带宽:

    ethtool 网卡名

    查看路由表

    route -n

    查看所有已经建立的连接:

    netstat -antp

    某端口使用情况:

    lsof -i:端口号
    netstat -apn|grep 端口号

    查看系统所有用户:

    cut -d: -f1 /etc/passwd

    查看当前使用的拥塞算法

    sysctl net.ipv4.tcp_available_congestion_control
    lsmod | grep bbr

  2. 优化TCP连接数[/etc/sysctl.conf]

    修改/etc/sysctl.conf

    vi /etc/sysctl.conf

    使配置生效

    /sbin/sysctl -p

  3. 调整最大文件打开数

    vim /etc/security/limits.conf

    vim /etc/security/limits.d/20-nproc.conf

    或者加入开机启动每次自动设置
    vim /etc/rc.local

  4. 调整字符集，使其支持中文

    locale -a 可通过grep过滤查看是否有中文语言包
    如果没有语言包

    yum install kde-l10n-Chinese
    yum reinstall glibc-common

    vim /etc/locale.conf
